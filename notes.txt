you will need:
git clone https://github.com/nlohmann/json

build & run:
g++ prompt.cpp -o prompt && ./prompt

deploy to access anywhere:
cp ./prompt ~/bin/prompt

build & deploy & run:
g++ prompt.cpp -o prompt && cp ./prompt ~/bin/prompt && prompt

To build and run the tests:
cd tests && g++ tests.cpp -o tests && ./tests && cd ..

All in one:
Build and run tests then build and deploy and run the prompt:
cd tests && g++ tests.cpp -o tests && ./tests && cd .. && g++ prompt.cpp -o prompt && cp ./prompt ~/bin/prompt && prompt
